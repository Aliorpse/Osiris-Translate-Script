# Osiris-Translate-Script
Osiris | 开源Counter-Strike 2作弊软件 - 自动翻译脚本

# 使用方法
将 `app.js` 放在 `Osiris` 根目录下,命令行运行 `node app` <br>
(需要一个JavaScript运行环境,我推荐使用[node.js](https://nodejs.org/zh-cn),当然,不一定要这个)<br>
替换完后再构建一次 Osiris 得到的就是翻译过的了<br>
